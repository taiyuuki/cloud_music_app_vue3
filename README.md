
# 说明

Vue3 + ts 练习项目，模仿网易云的移动端音乐播放器。

接口来自[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

目前实现的功能：

* 首页
  * 轮播图
  * 获取热门歌单
* 登录
  * 获取个人信息
  * 获取个人歌单
* 搜索
  * 根据输入推荐搜索
  * 保存搜索历史（10个）
* 播放功能
  * 播放控件
  * 全屏播放
  * 播放列表
  * 歌词平滑滚动
  * 黑胶唱片动画
